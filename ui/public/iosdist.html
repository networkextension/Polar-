公开分享页

每个应用都有一个 App Store 风格的公开页，开关打开后任何人扫码或点链接都能看到应用、点击安装、提交测试申请（自动邀请到上面绑定的 Beta Group）。所有公开应用同时会出现在 App 广场 索引页。

在新窗口预览
测试申请收件箱

公开页提交的「申请测试」记录都在这里。已 ASC 配置时会自动发出邀请邮件；状态 pending 表示尚未自动处理（缺 ASC 配置或缺 Beta Group 绑定），可在「ASC 集成」面板手动邀请。

    App Store Connect 测试组绑定

    把当前应用映射到 ASC 中的某个 App + 默认 Beta Group，之后用邮箱邀请测试者会自动加进这个组。
    ASC API Key 是 账号级的（对当前平台账号下所有应用通用），在右上角「ASC API Key」按钮配置；如管理多个 Apple 开发者账号，请用不同平台账号分别登录。

    尚未配置 ASC API Key，请先在右上角配置。
    邮件邀请测试者
    TestFlight 邀请链接

    如果你已经在 App Store Connect 配置好 TestFlight 公测组，把邀请链接（如 https://testflight.apple.com/join/XXXXX）粘贴到这里。OTA 不可装的 App Store 类型版本会优先引导用户走 TestFlight。
